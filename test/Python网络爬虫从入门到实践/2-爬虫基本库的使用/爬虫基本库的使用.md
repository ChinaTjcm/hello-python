## 一、Chrome 抓包详解

        抓包是抓取客户端与远程服务器通信时传递的数据包。Chrome 内置了一套功能强大的
    Web 开发和调试工具，可用来对网站进行调试和分析，用来抓包非常方便。可以通过下面
    的几种方式打开开发者工具。
    （1）在 Chrome 界面按 F12 键。
    （2）右键单击网页空白处，单击检查。
    （3）打开谷歌【开发者工具】
